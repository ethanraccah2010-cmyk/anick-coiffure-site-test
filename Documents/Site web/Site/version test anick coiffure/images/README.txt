DOSSIER IMAGES — Anick Coiffure
================================

Déposez ici les photos du salon. Tant qu'une image est absente, le site
affiche automatiquement un placeholder doux (pas d'image cassée).

Fichiers attendus (vous pouvez les ajouter au fur et à mesure) :

  hero.jpg      → grande photo du salon (hero d'accueil), format portrait 4:5 idéal
  salon-1.jpg   → Anick en train de coiffer / ambiance (portrait 3:4)
  salon-2.jpg   → détail coiffure / intérieur (portrait 3:4)
  real-1.jpg    → réalisation 1 (galerie + accueil)
  real-2.jpg    → réalisation 2
  real-3.jpg    → réalisation 3
  real-4.jpg    → réalisation 4
  real-5.jpg    → réalisation 5
  og.jpg        → image de partage réseaux sociaux (1200 x 630 px recommandé)

Conseils :
  - Formats acceptés : .jpg / .webp (pensez à mettre à jour le src dans le HTML
    si vous changez d'extension).
  - Compressez vos images (ex. squoosh.app) pour un chargement rapide.
  - Gardez des noms de fichiers EXACTEMENT identiques à ceux ci-dessus pour
    qu'ils s'affichent sans modifier le code.
